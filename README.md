## 1. 리눅스 명령어에서 top, ps, jobs, kill 명령어를 조사하기
|명령어|설명|
|---|---|
|top|현재 OS의 상태를 나타내주는 CLI 어플리케이션|
|ps|현재 실행중인 프로세스의 목록을 보는 명령어|
|jobs|작업의 상태를 표시하는 명령어|
|kill|프로세스에 특정한 시그널을 보내는 명령어|
